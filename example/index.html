sequential shots

A carousel web component written in vanilla javascript.

Usage

Just put the images and videos in the <sequential-shots> tag and you've got it.

Accessibility

The component accessible from the keyboard (arrow keys and M key to switch sound). You can focus it by clicking on it or using the tab key. It also supports versions with higher contrast and reduced motion.

Installation

If you want to try it, just copy this script into your project or install it with npm:

npm install dmkskn/sequential-shots

It's already working fine, but keep in mind that development is still ongoing. Things can change.
